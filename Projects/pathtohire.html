PathToHire

AI-powered mock interview platform that generates personalized technical questions. Democratizes interview coaching through adaptive generative AI.

Flask · RapidAPI Backend
LLMs Generative AI

PathToHire is an AI-powered mock interview platform that generates personalized technical questions based on the candidate’s profile and target role. It aims to democratize interview coaching by providing adaptive, generative AI–driven practice.

The backend is built with Flask and integrates with RapidAPI for AI capabilities. The project demonstrates how LLMs can be used to create practical, human-centered tools for career development.
