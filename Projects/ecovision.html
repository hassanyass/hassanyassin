    Computer Vision & Sustainability

    EcoVision AI

    Intelligent Waste Sorting Ecosystem for Smart Cities

    Python Flask OpenCV YOLOv8 MobileNetV2 TensorFlow

    The Problem

    Urban waste management is inefficient, expensive, and currently relies on human workers performing hazardous manual labor in unsanitary conditions.

    While computer vision has the potential to automate this, real-world waste facilities present complex challenges: overlapping items, poor lighting, and dirty materials that confuse standard AI models.

    The Solution

    EcoVision AI is a fully integrated, web-based intelligent waste recognition system. It bridges the gap between complex computer vision models and practical environmental management.

    Designed for deployment on energy-efficient edge hardware (like the NVIDIA Jetson Nano), it turns ordinary recycling bins into interactive "Smart Bins."

    The ML Pipeline

    To ensure industrial-grade robustness, I architected a multi-stage ML pipeline:

    Advanced Preprocessing

    Utilizing Contrast Limited Adaptive Histogram Equalization (CLAHE) and Canny edge detection to force the model to focus on structural geometry rather than color or lighting artifacts.

    Dual-Engine Architecture

    1. MobileNetV2 engine focuses on precise validation of individual items to ensure high-purity recycling streams.
    2. YOLOv8-based inventory mode identifies and tallies multiple overlapping waste items within complex scenes.

    Key Metrics

    87.53%

    Accuracy in classifying five major waste categories (Aluminium, Carton, Glass, Paper, Plastic) using MobileNetV2

    93.00%

    Precision reached on transparent Glass items, a notoriously difficult material for AI to classify

    0.9918

    mAP@50 achieved by the YOLOv8n multi-object detection model for high-throughput inventory tracking

    User Interface & Future Impact

    Responsive Flask GUI

    The system features a Flask-based graphical user interface with a responsive dashboard and drag-and-drop functionality. This enables non-technical stakeholders to conduct real-time sorting and inventory tracking without writing a single line of code.

    Scalable Smart City Solution

    This technology lays the foundation for scalable, AI-powered waste-sorting ecosystems that improve recycling efficiency, protect human workers, and support the circular economy in smart cities.

    EcoVision AI

    Intelligent waste sorting ecosystem using computer vision and deep learning. Transforms recycling bins into interactive Smart Bins with 87.53% accuracy.

    87.53% Accuracy
    YOLOv8 · MobileNetV2 Tech Stack

    EcoVision AI is an intelligent waste sorting ecosystem that combines computer vision and deep learning to transform traditional recycling bins into interactive Smart Bins. The system achieves 87.53% accuracy in classifying and sorting waste materials in real time.

    The project leverages YOLOv8 for object detection and MobileNetV2 for efficient classification, with a Flask backend serving the model. It demonstrates how AI can be deployed at the edge for environmental impact—reducing contamination in recycling streams and encouraging better disposal habits.

    Technologies used include Python, OpenCV, YOLOv8, MobileNetV2, and Flask. The repository is available on GitHub for exploration and collaboration.
