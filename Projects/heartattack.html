Heart Attack Prediction

Real-Time Cardiac Risk Assessment powered by Deep Learning

Python Flask Deep Learning Genetic Algorithms Fuzzy Logic Scikit-Learn

The Problem

Cardiovascular disease is the leading cause of death globally, but many fatal incidents could be prevented with early detection.

Unfortunately, advanced diagnostic tools are often inaccessible to the average person, and subtle warning signs in standard health metrics are easily missed until it is too late.

The Solution

A lightweight, accessible early-warning system. I developed a web application that democratizes diagnostic insights.

By inputting standard, easily attainable health metrics (like blood pressure, cholesterol, and age), users receive a real-time risk assessment powered by top-tier computational intelligence models, encouraging preventative care.

The Data Science Pipeline

Instead of blindly applying a single model, I engineered and benchmarked four distinct Computational Intelligence architectures to find the absolute best fit for medical data:

Feature Engineering

Cleaned and processed raw medical datasets to optimize predictive performance.

Model Benchmarking

Developed an Artificial Neural Network (ANN), a Deep Neural Network (DNN), a Hybrid Genetic Algorithm-ANN (GA-ANN), and a Fuzzy Inference System (FIS).

Rigorous Evaluation

Conducted comparative analysis using Precision, Recall, and F1-score to minimize false negatives (which are fatal in medical diagnoses).

Key Metrics

89%

Accuracy in predicting heart attack risk using the top-performing Deep Learning architecture

Zero-Latency

Inference deployed via Flask to provide instant risk assessment upon user input

Engineering Insight

This project highlights the intersection of classical AI (Fuzzy Logic) and modern Deep Learning (DNNs). The use of Genetic Algorithms to optimize the Neural Network demonstrated how bio-inspired computing can solve complex medical classification problems.
