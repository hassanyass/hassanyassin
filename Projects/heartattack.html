Heart Attack Prediction

Real-time cardiac risk assessment using Deep Neural Networks and Genetic Algorithms. Achieves 89% accuracy with zero-latency inference for preventative healthcare.

89% Accuracy
DNN · Genetic Algorithms Tech Stack

This project implements real-time cardiac risk assessment using Deep Neural Networks (DNN) and Genetic Algorithms for feature selection and model optimization. It achieves 89% accuracy and is designed for zero-latency inference to support preventative healthcare applications.

The system is backed by a Flask API for easy integration. It demonstrates how deep learning and evolutionary algorithms can be combined for robust, interpretable medical risk modeling.
