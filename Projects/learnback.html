LearnBack

AI learning platform where students teach concepts to the system. Detects misconceptions and provides targeted feedback for deeper understanding.

Generative AI · NLP Tech Stack
React · Three.js Frontend

LearnBack is a reverse-tutoring platform where students teach concepts to an AI system. The system detects misconceptions and provides targeted feedback to support deeper understanding. The frontend was built with React and Three.js (Student Tutoring Website repo).

The project combines generative AI and NLP to create an interactive, feedback-driven learning experience that reinforces understanding through explanation and correction.
